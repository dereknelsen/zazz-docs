---
title: Tabs
description: A segmented tab control driven entirely by grouped radio inputs, with no JavaScript.
---

Tabs are CSS-only: grouped `input[type="radio"][role="tab"]` inputs drive the panels and the
sliding indicator. **Panel order must match the radio order.** Lay them out vertically with
`data-direction`.

## API

| Attribute        | Where   | Values                             |
| ---------------- | ------- | ---------------------------------- |
| `data-direction` | `.tabs` | `vertical` (horizontal is default) |
