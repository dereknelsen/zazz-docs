**Zazz is three CSS files (variables, reset, utilities), a Figma kit, and a Webflow kit. All share the same names.**

Drop the CSS into your project, import it once at the entry point, and every page below picks up the system. No JavaScript dependencies, no build step, no opinions about your framework.

Setup differs by platform. Pick where you're starting.
