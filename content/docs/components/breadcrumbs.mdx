---
title: Breadcrumbs
description: Hierarchical navigation composed from link buttons and separators.
---

Breadcrumbs are a composition, not a dedicated primitive. A `<nav>` wraps a `<menu>` of
link buttons (`.button[data-variant="link"]`) separated by muted `/` glyphs. Mark the current
page with `aria-disabled="true"` so it reads as the active, non-interactive crumb.
