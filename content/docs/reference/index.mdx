The Reference section documents the data: every token, every utility class, every default value. One page per design-system property (colors, padding, box-shadow, border-radius) with values, code examples, and customization patterns.

If you're looking for the _system_ (why tokens are layered, how modes work, how primitives nest), start with [Foundations](/docs/foundations). Reference is the lookup table; Foundations is the manual.
