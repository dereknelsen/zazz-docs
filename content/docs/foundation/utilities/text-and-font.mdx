These are the typographic helper utilities in `_utilities.css`: alignment, casing, font family,
and fine-grained text-flow controls. The fluid **size scale** (`text-h1`, `text-sm`, …) and
**weight** utilities live in the type system (see
[Typography in the overview](/docs/getting-started/overview#typography)), and [text color](/docs/foundation/utilities/text-color)
covers the color roles.

## Text align

| Class          | Declaration           |
| -------------- | --------------------- |
| `text-left`    | `text-align: left`    |
| `text-center`  | `text-align: center`  |
| `text-right`   | `text-align: right`   |
| `text-justify` | `text-align: justify` |

All four have responsive variants, e.g. `text-center @md:text-left`.

## Text transform

| Class         | Declaration                  |
| ------------- | ---------------------------- |
| `uppercase`   | `text-transform: uppercase`  |
| `lowercase`   | `text-transform: lowercase`  |
| `capitalize`  | `text-transform: capitalize` |
| `normal-case` | `text-transform: none`       |

<Callout type="info">
  House style is **sentence case** for UI text: headings, buttons, links, labels, and nav. Reach for
  `uppercase` only when a design genuinely calls for it (the eyebrow style already handles its own
  caps).
</Callout>

## Font family

Three stacks, mapped to the family tokens.

| Class        | Token               |
| ------------ | ------------------- |
| `font-sans`  | `var(--font-sans)`  |
| `font-serif` | `var(--font-serif)` |
| `font-mono`  | `var(--font-mono)`  |

## Text wrapping & overflow

Modern text-flow controls for cleaner typesetting and truncation.

| Class          | Declaration                          | Use                                     |
| -------------- | ------------------------------------ | --------------------------------------- |
| `text-balance` | `text-wrap: balance`                 | even line lengths in short headings     |
| `text-pretty`  | `text-wrap: pretty`                  | avoid orphans in body copy              |
| `leading-none` | `line-height: 1`                     | tight, single-line labels               |
| `tabular-nums` | `font-variant-numeric: tabular-nums` | aligning figures in tables and counters |

### Line clamp

Truncate to a fixed number of lines with an ellipsis. `line-clamp-1` … `line-clamp-6` set the
line count.

| Class          | Lines |
| -------------- | ----- |
| `line-clamp-1` | 1     |
| `line-clamp-2` | 2     |
| `line-clamp-3` | 3     |
| `line-clamp-4` | 4     |
| `line-clamp-5` | 5     |
| `line-clamp-6` | 6     |

```html
<h3 class="text-h3 text-balance">A headline that wraps onto evenly balanced lines</h3>
<p class="text-sm text-muted-foreground line-clamp-2">
  A two-line teaser that truncates with an ellipsis once it runs past the second line…
</p>
```
