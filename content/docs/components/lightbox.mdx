---
title: Lightbox
description: An Embla-powered image gallery with a fullscreen carousel dialog and thumbs.
---

The lightbox pairs an inline Embla carousel with a fullscreen `<dialog>` carousel and a
thumb strip. Slides open the modal via `command="show-modal"`, and `data-embla-*`
attributes configure the carousels entirely in markup. It needs the Embla scripts, which this
preview loads automatically.

## Default

<Preview src="lightbox/default" />

Carousel behavior comes from `data-embla` role attributes (`root`, `viewport`, `container`,
`slide`, `thumbs`) and `data-embla-*` options (`data-embla-loop`, `data-embla-classnames`, …)
— there is no JavaScript to write.
