    Zazz framework

    Test pages

    Standalone pages exercising the container band system, responsive utilities, components, and form fields. Built with nothing but Zazz classes. Resize to watch them respond.

    Layout

    Container bands

    The subgrid band system: bands from xs to bleed, per-child data-container, a data-container default, the responsive @md:container variants, and the article reading width.

    Utilities

    Responsive utilities

    The @xs: through @xl: prefixes across grid columns, flex direction, display toggles, column spans, basis, and alignment.

    View page
    Components

    Component kitchen sink

    Buttons, badges, breadcrumbs, cards, avatars, accordion, tabs, tooltip, dropdown, dialog, and prose, adapted from each component's source.

    View page
    Forms

    Fields and validation

    Inputs, input groups, the password toggle, textarea, select, checkboxes, radios, switches, and the slider, with :user-invalid validation.

    View page
    Tip

    No build step

    These pages load zazz.css directly and are served raw at /zazz/components/tests/. Plain HTML plus Zazz tokens, utilities, and components.
